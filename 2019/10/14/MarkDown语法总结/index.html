MarkDown语法总结

终于开始去写博客了，要学习下MarkDown语法，第一篇博客就用来整理，记录常用的MarkDown语法，方便以后的查询，使用。

MarkDown语法主要分为以下几个部分：标题，强调，引用，分割线，列表等。

标题

功能：表示各级标题（最多有6级标题）

公式和说明：

#+空格+文字

示例：

# 我爱MarkDown

## 我爱MarkDown

### 我爱MarkDown

效果：

我爱MarkDown

我爱MarkDown

我爱MarkDown

注意：如果中间不加空格，在本地的MarkDown编辑器上可以正确显示标题，但同步到网站上，则不能正确显示

强调

功能：文字的倾斜，加粗，倾斜和加粗，加删除线。

公式和说明：

*+文字+*：文字倾斜

_+文字+_：文字倾斜

**+文字+**：文字加粗

__+文字+__：文字加粗

***+文字+***：文字加粗加斜

___+文字+___：文字加粗加斜

~~+文字+~~：文字加删除线

示例：

*我爱MarkDown* 或 _我爱MarkDown_

**我爱MarkDown** 或 __我爱MarkDown__

***我爱MarkDown*** 或 ___我爱MarkDown___

~~我爱MarkDown~~

效果：

我爱MarkDown

我爱MarkDown

我爱MarkDown

我爱MarkDown

引用

功能：就只是引用吧

公式和说明：

>+文字

示例：

>我爱MarkDown

>>我爱MarkDown

>>>我爱MarkDown

效果：

我爱MarkDown

我爱MarkDown

我爱MarkDown

分割线

功能：添加分割线，分割功能

公式和说明：

符号 - 或 * ，3个及以上

示例：

-–

-—

-—-

-—–

***

****

*****

******

效果：

链接

功能：添加网站链接，图片链接等

公式和说明：

[自定义名称](url “自定义名称(可选的)”)：网站链接

示例：

[我的github](https://zhangnanyue.github.io)

效果：

我的github

我的github

[我的github1][1]

我的github2
